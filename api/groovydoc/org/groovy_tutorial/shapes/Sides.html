Package: org.groovy_tutorial.shapes

[Groovy] Trait Sides

    • org.groovy_tutorial.shapes.Sides

    • A basic trait describing the outer edges (sides) of a 2D shape. This trait uses the missingProperty method to allow implementations to define sides using lower case characters (e.g. a, b, c). These properties must:

      • Be a single, lower-case character (matching Sides.SIDE_NAME_PATTERN)
      • Be only assigned a positive numeric value
      Once getPerimeter is called, the sides map is locked down and can't be modified.
